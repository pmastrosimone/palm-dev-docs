8.33.5.3 longcalls

The longcalls directive enables or disables function call relaxation. See section Function Call Relaxation.

 
    .begin [no-]longcalls
    .end [no-]longcalls

Call relaxation is disabled by default unless the `--longcalls' command-line option is specified.
