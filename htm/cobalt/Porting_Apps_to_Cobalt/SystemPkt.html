115 SystemPkt.h

Deleted APIs

      

The system packet APIs, used in earlier Palm OS releases by the debugger, console, and remote UI modules when communicating (via the Serial Link Manager) with the host computer, are not supported in Palm OS Cobalt.

Note that these APIs were never documented, and thus were not intended for use by third-party developers.
