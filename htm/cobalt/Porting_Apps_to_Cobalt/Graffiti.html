51 Graffiti.h

Deleted APIs

Unchanged APIs

      

Prior to Palm OS Cobalt applications could access the Graffiti® and Graffiti 2 handwriting recognition engines directly. This is no longer the case in Palm OS Cobalt. Accordingly, except for the Graffiti-related feature constants, all of the APIs formerly declared in Graffiti.h will not be found in the Palm OS Cobalt headers.
