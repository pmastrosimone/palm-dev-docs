101 SerialSdrv.h

Deleted APIs

      

Under Palm OS Cobalt the Serial Manager is implemented as a STREAMS driver and a compatibility library that lets you continue to use the Serial Manager API. The Serial Driver APIs listed here, that were used in previous versions of Palm OS, are no longer supported.
