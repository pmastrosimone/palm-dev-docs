100 SerialMgrOld.h

Deleted APIs

Unchanged APIs

      

Prior to Palm OS Cobalt there was both the "Old Serial Manager" and the "Serial Manager." The "Old Serial Manager" is not supported in Palm OS Cobalt. Applications should use the Serial Manager instead.
