8.23.1.4 Machine Model Options

These options enable the instruction set extensions supported by a particular machine model, and disables all other extensions.

-m11/03
Same as -mkd11f.

-m11/04
Same as -mkd11d.

-m11/05 | -m11/10
Same as -mkd11b.

-m11/15 | -m11/20
Same as -mka11.

-m11/21
Same as -mt11.

-m11/23 | -m11/24
Same as -mf11.

-m11/34
Same as -mkd11e.

-m11/34a
Ame as -mkd11e -mfpp.

-m11/35 | -m11/40
Same as -mkd11a.

-m11/44
Same as -mkd11z.

-m11/45 | -m11/50 | -m11/55 | -m11/70
Same as -mkb11.

-m11/53 | -m11/73 | -m11/83 | -m11/84 | -m11/93 | -m11/94
Same as -mj11.

-m11/60
Same as -mkd11k.
