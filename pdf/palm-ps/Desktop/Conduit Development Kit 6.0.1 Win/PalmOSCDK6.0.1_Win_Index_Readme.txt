The CDK document set is provided in Acrobat PDF format, which requires 
Adobe Reader 6.0 or later.  To get the free Adobe Reader, go to 

   http://www.adobe.com/products/acrobat/readstep2.html

The CDK document set includes a PDF index, which enables you to perform 
fast, full-text searches across all the documents in the set.  To use the 
PDF index, simply double-click the file 

   PalmOSCDK6.0.1_Win_Index.pdx

This starts Adobe Reader and presents you with search options.  Enter 
your search criteria in the search box and click the search button.  A 
list of CDK documents matching your criteria is displayed, with links to 
each matching instance within each document.
